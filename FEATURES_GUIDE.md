## ðŸ“Š Data Management

### Adding a New Report
1. Open `data/reports.json`
2. Copy an existing report object
3. Update all fields:
   - `id`: Unique identifier (lowercase-with-dashes)
   - `name`: Display name
   - `reportingDomain`: Category
   - `processOwner`: Owner name
   - `reportingFrequency`: Daily/Weekly/Monthly/Quarterly/Annual
   - `shortDescription`: 1-2 sentences
   - `detailedDescription`: Full paragraph
   - `features`: Array of feature strings
   - `accessInstructions`: How to access
   - `contactEmail`: Contact email
   - `lastUpdated`: YYYY-MM-DD format
4. Save file
5. Report appears automatically!

### Updating Existing Report
1. Find report by `id` in `data/reports.json`
2. Update desired fields
3. Save file
4. Changes appear immediately

### No Database Required!
- All data in one JSON file
- Easy version control with Git
- Simple backup (just copy the file)
- Fast performance (static generation)
